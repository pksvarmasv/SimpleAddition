Visual Math

Main objective of this project is to help visually impaired individuals learn mathematics.
The idea is to present the concepts in a screen-reader friendly way for easy learning experience.



  • Simple Addition
  • SimpleSubtraction
  • SimpleMultiplication
  • Polynomial Division
  • Synthetic Division
  • Matrix Addition
  • Decimal To Binary Conversion
  • Binary To Decimal Conversion
